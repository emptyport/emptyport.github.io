Bioinformatics is just code for scientific programming.

I consider myself fairly fluent in both JavaScript and Python. Python is an obvious choice for bioinformatics, but I mostly stick with JavaScript because 1) I like it and 2) I think it is an important language to learn/use as more and more software moves to the cloud.

One of my pet projects is Omics.js, a collection of JavaScript modules that are useful to the "omics." Right now the focus is on proteomics because that is what I do, but the goal is to include the other omics as well. Omics.js is my way of compiling a list of good JavaScript modules and to see what is still missing. I am slowly filling in the gaps with my own modules that you can see on my Github or on npm.
